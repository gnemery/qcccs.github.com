Welcome to the CSCCS207 Github Page!

CSC 207 is computer science class at Quinsigamond Community College that specializes in programming.

We work on and create many open source project every semester. This semester, Spring 2013, we are working on the following projects:

  • FireRecord

    A plugin for firebug which can be used to record your actions and play them back.

    Developers: Peter Atechian, David Westgate, Rylan Doherty, Bill Jellesma

  • CppPalette

    A C++ palette plugin for netbeans.

    Developer: Peter Atechian

  • PHPPalette

    A PHP palette plugin for netbeans.

    Developer: Bill Jellesma

  • JavascriptPalette

    A Javascript palette plugin for netbeans.

    Developer: Tim Mesite

  • JavaPalette

    A Java palette plugin for netbeans.

    Developer: Ian Hickey

We also have a list of projects that many of us have worked on in the past. These are the projects that many of us have worked on:

  • Java Chat

    A Chat program that will make use of a local database. WAMPserver or another localhost must be installed

    Developer: Bill Jellesma

  • AutoExplorer

    A program to record one's interactions on the internet in the Ruby language

    Developers: Ian Hickey, Khem Bastola, Naishadh Patel, James Ta, John Donas, Chris Mitchell, Em Trieu, Jeremiah T. Johnson, Naishadh Amin, Julion DeVincentis
